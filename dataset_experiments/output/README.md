Generated by scripts in `dataset_experiments/scripts/`. Run individually from the project root, e.g.:

```bash
python dataset_experiments/scripts/explore_keys.py
python dataset_experiments/scripts/explore_chords.py
python dataset_experiments/scripts/explore_raw_chords.py
```

Hooktheory viewers (`explore_hooktheory.py`, `explore_raw_hookpad.py`) only inspect
songs that are in `data/cache/hooktheory` — the same filter as
`scripts/convert_data_to_cache/convert_hooktheory_to_cache.py`: **MELODY + HARMONY tags, no TEMPO_CHANGES**.
Alignment tags are not part of that filter. See `scripts/hooktheory_cache_filter.py`.

Use matching `--split` and `--song-index` (default: `train`, `7`) so cache JSON,
MIDI, tokens, and raw Hookpad JSON refer to the same song.

| Subfolder | Script | Contents |
|-----------|--------|----------|
| `hooktheory/` | `explore_hooktheory.py` | Sample cache song JSON, MIDI, tokenized tensors |
| `raw_json/` | `explore_raw_hookpad.py` | Raw Hookpad JSON for the same cache song |
| `keys/` | `explore_keys.py` | Key/mode statistics for all cache-eligible songs |
| `chords/` | `explore_chords.py` | Chord counts in Hooktheory interval form (+ inversion) |
| `raw_chords/` | `explore_raw_chords.py` | Hookpad-native chords; voicing reconstruction inventory |
| `chord_midi_coverage/` | `explore_chord_midi_coverage.py` | Chord → token → RealJam MIDI pitch coverage |
| `pop909/` | `explore_pop909.py` | Chord symbol parsing examples |
| `wjazzd/` | `explore_wjazzd.py` | DB schema (+ optional CSV export) |

Each subfolder contains a `run_summary.txt` when the script succeeds.
